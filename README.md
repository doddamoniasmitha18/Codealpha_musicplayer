🎵 Music Player

A simple, responsive, and interactive music player built using HTML, CSS, and JavaScript. Play, pause, and navigate through your favorite songs with an intuitive interface.

🛠️ Features

Play and pause music tracks

Next and previous song navigation

Display of song title and artist

Progress bar with current time and duration

Album cover display

Responsive design for all screen sizes

💻 Technologies Used

HTML5 – Structure of the player

CSS3 – Styling and responsive design

JavaScript – Audio controls and dynamic updates
